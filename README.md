Morsecode
=============

Converts plain text to morsecode.

Usage
-----
    $echo “hello world” | ./morse.sh
    ...././.-../.-../—-//.—/—-/.-./.-../-..

Letter spacing = `/´

Word spacing = `//´

Installation
------------
Place the two files 'morse.sh' and 'alphabet-morse.sed' in the same folder somewhere in your $PATH.
